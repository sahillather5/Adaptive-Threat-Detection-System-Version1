# 📌 Project Overview

The **Behavior-Based Login Monitoring System (Version 1)** is designed to enhance traditional login security by monitoring user login behavior on the **main screen** of a computer system.

Instead of relying only on passwords, this system observes login attempt patterns and identifies abnormal behavior that may indicate unauthorized access attempts.

This project follows behavior-based security principles commonly used in **Security Operations Center (SOC)** environments and modern **User and Entity Behavior Analytics (UEBA)** systems.

---

# 🎯 Objectives

The primary objectives of this system are:

- Monitor login attempts on the main screen
- Identify repeated failed login attempts
- Detect unusual login behavior patterns
- Generate alerts for suspicious activity
- Improve login security using behavioral monitoring
- Provide early warning against unauthorized access attempts

# 🔔 Alert Mechanism

The system provides **real-time alert notifications** when suspicious login behavior is detected.

Alerts may be triggered in situations such as:

- Multiple incorrect login attempts
- Repeated failure patterns
- Behavior deviating from normal usage patterns
- High-risk login activity conditions

Alerts are displayed through **on-screen notifications**, enabling quick user awareness.

---

# 📊 Logging and Monitoring

All login activity events are recorded in structured logs.

These logs are used to:

- Monitor login history
- Track failed attempts
- Observe behavior trends
- Support security analysis
- Assist in identifying suspicious activity

Logging plays a key role in behavior-based security monitoring.

---

# 🏗️ System Workflow Overview

The simplified workflow of the system:

1. User attempts login on main screen  
2. Login attempt is recorded  
3. System observes behavior pattern  
4. Activity is evaluated against normal usage behavior  
5. Suspicious patterns trigger alerts  
6. Logs are stored for monitoring and analysis  

Note: Internal detection logic is intentionally abstracted to maintain system integrity.
